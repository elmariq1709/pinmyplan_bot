# ⚡ БЫСТРЫЙ СТАРТ - 5 МИНУТ

## 1️⃣ Получи токен (2 минуты)

```
Telegram → поиск @BotFather → /newbot 
→ дай имя → дай username (заканчивается на _bot)
→ скопируй токен (вида 123456:ABC...)
```

## 2️⃣ Подготовь файлы (1 минута)

Скопируй в папку:
- `planner_bot.py`
- `database.py`
- `config.py`
- `requirements.txt`

Создай `.env` файл с содержимым:
```
BOT_TOKEN=ВСТАВЬ_СВОЙ_ТОКЕН_СЮДА
DATABASE_PATH=planner.db
```

## 3️⃣ Установи Python (проверь наличие)

```bash
python --version  # Должно быть 3.10+
```

Если нет - скачай с https://www.python.org

## 4️⃣ Установи зависимости (1 минута)

```bash
pip install -r requirements.txt
```

## 5️⃣ Запусти бота! (30 секунд)

```bash
python planner_bot.py
```

Если видишь `🚀 Бот запущен!` → ✅ РАБОТАЕТ!

## Первый тест

1. Найди бота в Telegram (по username)
2. Отправь `/start`
3. Нажми `➕ Новая задача`
4. Добавь задачу "Тест"
5. Нажми `📋 Мои задачи`

✅ Всё работает? Переходи к деплою!

---

## Деплой на облако (5 минут)

### Render (самый простой)

1. GitHub репозиторий → код
2. Render.com → New Web Service
3. Выбери репозиторий
4. Environment:
   - BOT_TOKEN = твой токен
   - DATABASE_PATH = planner.db
5. Deploy!

**Готово! 🎉**

## 🚨 Что-то не работает?

1. **"No module named telegram"**
   → `pip install python-telegram-bot`

2. **"BOT_TOKEN is None"**
   → Создай `.env` с токеном

3. **"Ошибка при запуске"**
   → Проверь Python 3.10+

4. **Бот не отвечает**
   → Проверь что токен скопирован правильно
