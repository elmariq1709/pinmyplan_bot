# 📋 Telegram Planner Bot

Полнофункциональный бот-планнер для Telegram с поддержкой категорий, приоритетов, сроков и напоминаний.

## ✨ Возможности

- ✅ Добавление, удаление и завершение задач
- 🏷️ Категоризация (работа, дом, учеба, здоровье, личное, покупки)
- ⚡ Приоритеты (высокий, средний, низкий)
- 📅 Установка сроков и времени
- 📝 Описание для каждой задачи
- 📊 Статистика и просмотр задач на сегодня
- 💾 Надежное хранилище SQLite

## 🚀 Быстрый старт

### 1. Получи токен
Найди в Telegram `@BotFather`, создай нового бота и скопируй API Token.

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой переменные окружения
```bash
# Создай .env файл
cp .env.example .env

# Отредактируй .env и вставь свой токен:
# BOT_TOKEN=123456:ABC-DEF...
```

### 4. Запусти бота
```bash
python planner_bot.py
```

## 📱 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/add` | Добавить новую задачу |
| `/tasks` | Все активные задачи |
| `/today` | Задачи на сегодня |
| `/status` | Статистика |
| `/complete [ID]` | Завершить задачу |
| `/delete [ID]` | Удалить задачу |
| `/help` | Справка |

## ☁️ Деплой

### Render (рекомендуется)
1. Загрузи код на GitHub
2. Перейди на [render.com](https://render.com)
3. Создай новый Web Service из репозитория
4. Добавь `BOT_TOKEN` в Environment variables
5. Деплой готов! 🚀

### Railway
1. Загрузи код на GitHub
2. Перейди на [railway.app](https://railway.app)
3. Создай новый проект из GitHub
4. Добавь `BOT_TOKEN` в переменные окружения
5. Railway автоматически прочтёт Procfile и запустит бота

## 🐛 Решение проблем

**Бот не запускается?**
- Проверь что установлены все зависимости: `pip install -r requirements.txt`
- Убедись что BOT_TOKEN в `.env` правильный

**БД ошибки?**
- Удали `planner.db` и запусти бота снова

**На Render не работает?**
- Проверь логи в Render console
- Убедись что BOT_TOKEN добавлен в Environment variables
